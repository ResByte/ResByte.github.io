Projects

1. SLAM backend for cloud based mapping

Real-time mapping is still a challenge for a lot of mobile robots, as available computation power is very limited and also to avoid network overflow from rgbd data, it is necessary to rethink algorithms. In this project, a keyframe-graph based map is created and a metric distance based loop closure detection is applied. Further, the constructed graph is is optimized using g2o on regualar intervals to minimize the error cost. This is implemented in ROS using pcl, opencv and g2o. It is further fine tuned to perform on multi-core cpu.
